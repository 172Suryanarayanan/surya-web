Agro-based IoT and AI Project

In this project, I developed a smart agro-tech solution that combines IoT sensors and artificial intelligence to enhance agricultural productivity. The system utilized IoT devices to collect environmental data like soil moisture, temperature, and weather conditions using the Google Weather API. Using CNN-based detection models, the platform analyzed the data to provide insights and weather-based crop predictions. To further improve decision-making, I integrated reinforcement learning and built a ChatGPT-powered chatbot for interactive support. The project was developed using a full-stack architecture—Node.js and Flask on the backend, Firebase and Adafruit Cloud for data flow, and HTML/CSS/JavaScript for the frontend.



LULC-GDP Based Fund Utilization Analysis

This project focused on analyzing how land use and land cover (LULC) types affect regional fund allocation using GDP statistics. I designed a data-driven pipeline that merged satellite-based LULC patterns with economic indicators to identify inefficiencies in resource distribution. The analysis revealed how different land categories—urban, agricultural, and industrial—impacted the flow and effectiveness of government funding. This enabled an evidence-based approach to optimize fund utilization, highlight disparities, and guide policy improvements. The project leveraged Python libraries like Pandas and Matplotlib for data analysis and visualization.
